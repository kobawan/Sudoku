Rules

The classic Sudoku game involves a table of 81 squares. The table is divided into nine 3x3 grids, each containing nine squares.

To win the game, each of the nine grids has to contain all the numbers 1-9 within its cells. Each number can only appear once in a row, column or grid.

About

Hi! My name is Sara, or Kobawan in the interwebz. I've always been a fan of puzzles, so when I started programming, I came up with the idea to combine my two joys of puzzles and programming, to create this website.

It's been a fun project to create. Sometimes challenging and time-consuming, but in the end, a very good opportunity to learn programming.

Along with creating more projects than I can handle, I also knit, play with my dogs, and binge watch netflix.

I hope you enjoy!
